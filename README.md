## Run Locally

Clone the project

```bash
  git clone https://github.com/Sakthi-Sairam/Movie-webApp-1st-React-website
```

Go to the project directory

```bash
  cd Movie-webApp-1st-React-website
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
